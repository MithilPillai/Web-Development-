CSS Selectors

Applying CSS to different parts of HTML

1. The elements selector targets elements based on their HTML tag name.

  1. Class Selectors targets elements based on the value of the class attribute.
  2. ID Selectors targets elements based on the value of the id attribute.
  3. Attribute Selectors targets elements based on the value of the class attribute.
  4. Attribute Selectors targets elements based on their attributes and values.
